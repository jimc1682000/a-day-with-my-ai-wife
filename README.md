## 關於這個專案

這是一個互動式的 Terminal 風格網頁，記錄了我與 Claude Code 協作處理 Gitea OOM 事件的一天。

### 數據

- ⏱️ 7.5 小時的協作
- 💻 200+ 行程式碼
- 💡 14 個協作心法
- 🔧 7 次糾正

### 主題

- AI 系統的非確定性
- 上下文管理與記憶限制
- 人類與 AI 的協作模式
- 實際部署後驗證的重要性

## 操作方式

- **點擊** 或 **Space / Enter** — 進入下一個段落
- **滾輪** — 正常捲動瀏覽
- **Page Up / Page Down** — 跳到上/下一個章節

## 技術

純前端實作，無需後端：

- HTML5
- CSS3 (Terminal 風格)
- Vanilla JavaScript (打字動畫效果)

## 本地運行

```bash
python3 -m http.server 8080
# 或
npx serve .
```

然後開啟 http://localhost:8080
